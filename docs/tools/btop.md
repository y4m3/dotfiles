# btop

Concise notes focused on environment-specific behavior. See official docs for usage.

## Official Documentation

https://github.com/aristocratos/btop

## Installation

Managed by `run_once_280-utils-btop.sh.tmpl`:
- Installs binary to `~/.local/bin/btop`
- Configuration at `~/.config/btop/btop.conf`

## Environment-specific Configuration

- Theme: Tokyo Night
- Update interval: 2000ms (2 seconds)
- Rounded corners enabled
- Vim-style navigation enabled
